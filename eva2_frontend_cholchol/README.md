# Eva 2 Frontend - Rediseño Municipalidad de Cholchol

Proyecto académico de Desarrollo Frontend U2.

## Tecnologías
- HTML5 semántico
- CSS3 responsivo
- JavaScript DOM
- Bootstrap 5 vía CDN

## Cómo abrir
Abrir `index.html` en el navegador.

## Funcionalidades evaluativas
- 9 eventos de usuario implementados.
- Formulario con validación en tiempo real.
- Más de 5 modificaciones del DOM.
- Diseño responsivo, accesible y con navegación por teclado.
- Uso de patrón simple por funciones de renderizado y separación HTML/CSS/JS.
